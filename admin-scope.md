# Admin Scope

## Ziel
Pflege aller Inhalte der Website über ein simples, übersichtliches Dashboard.

## Bearbeitbare Bereiche
- Hero
- Aktuelles
- Kurse
- Preise
- Über mich
- Kontakt
- Settings
- Rechtliches

## Aktuelles
- optionale Sektionsüberschrift und Einleitung
- Liste von Themen: optionaler Kurztitel, Text, Bild (`url`, `alt`), Sortierung

## Hero
- title
- claim
- primary CTA label
- primary CTA url

## Kurse
- internal / external
- Titel
- Beschreibung
- Tag
- Uhrzeit
- Ort
- Buchungsstatus
- externalUrl bei external

## Preise
- Titel
- Preis
- Beschreibung
- Sortierung

## Über mich
- Titel
- Text
- Bild

**Hinweis:** Der **Kurz-„Über mich“-Teaser** direkt unter dem Hero ist aktuell **nicht** im Admin editierbar (fest in `AboutTeaserSection`). Textänderungen an der Person / Bio bitte parallel in **`about`** und in der Teaser-Komponente vornehmen, bis optionale Felder im Content-Modell ergänzt sind (siehe `content-model.md`).

## Kontakt
- E-Mail
- Telefon
- Formulartexte

## Settings
- Seitenname
- App-URL
- Logo-URL
- Meta Description
- OG Image URL

## Rechtliches
- Impressum
- Datenschutz

## UX-Anforderungen
- sehr clean
- geringe Komplexität
- klare Navigation
- Formulare mit guter Validierung
- keine unnötigen Features
